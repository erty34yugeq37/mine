CODE University Berlin – Personal Statement

School was fine, honestly. I did okay. But there was always this gap between how I actually figured things out — through doing, breaking, redoing — and what I was being asked to do in class. Sit, listen, summarise. I kept thinking there had to be somewhere that taught differently.

I came across CODE kind of by accident, through a video someone posted about their first semester. No lectures. They were just building stuff from week one. I watched it twice.

What CODE does with the Quest model is pretty simple on the surface — you pick a problem, you work on it, the learning follows from that — but I think most places that claim to be "project-based" still have a lecture series underneath it all. CODE doesn't seem to. The projects aren't a bolt-on. They're the whole thing. That's genuinely rare and it's the main reason I'm applying here and not somewhere else that sounds similar.

The three programmes — Software Engineering, Interaction Design, Product Management — not being siloed is also a big deal to me. I've been annoyed by how cleanly people separate those things. A product doesn't know which department made which bit. It just works or it doesn't. Training people to only ever see their slice of it seems like a bad idea, and CODE apparently agrees, given how the whole curriculum is set up around cross-disciplinary teams.

Berlin matters too, though it's hard to explain without sounding like I'm just name-dropping the city. What I mean is more specific than "Berlin is cool." CODE is in Neukölln, surrounded by people who are building startups, working in tech, making things — and those people aren't just career-fair guests at CODE, they're actually involved in how students learn. That's a different kind of proximity than most unis offer. You're not preparing to enter an industry. You're already sort of in it.

I don't have a perfect answer for what I'll build there. I'd be lying if I said I did. But I know how I learn, and I know that what CODE is doing matches it more closely than anything else I've found. That felt worth saying directly.
